# APIArticleExpressJS

Projet évalué.

## Initialisation

Installation des dépendences
```js
npm init -y; // Initialisation
npm install express sequelize mysql2 mangoose; // dépendences principales
npm install dotenv --save; // avoir un fichier .env
npm install winston; // création de logs
npm install uuid; // création d'id au format uuid
npm install --save-dev jest; // Pour les test

npm ci; // Pour "clear and install" toutes les dépendences
```
